# 🌌 Cosmic Evolution

Observe • Guide • Evolve

A science-inspired universe simulation where stars form, planets evolve, life emerges, and civilizations shape the cosmos.

Not a god simulator.
Not a sandbox with unlimited control.

Guide a living universe and discover what emerges.
